# 🏫 School Canteen – Frontend Task (Edzy)

This is a React-based frontend application for a school canteen digital ordering system.  
Students can view snacks, place orders, and track their spending.

---

## 🚀 Live Features

Live: https://aniledzyai.vercel.app/

## Screen-Recording

Link: https://drive.google.com/file/d/1uX9Jj-hQFZA9YiVvmc7BuDqF26Mg3X13/view?usp=sharing

### 🥪 Snacks Page
- View all available snacks
- See price and total orders count
- Place order via modal (select student + quantity)

### 👨‍🎓 Students Page
- View all students
- See referral code and total spending
- Add new students

### 📊 Student Detail Page
- View student details
- See all past orders
- Displays snack name, quantity, and total amount

## 🛠️ Tech Stack

- 🔁 React Query (TanStack Query)

- 🧾 React Hook Form

## ⚙️ Setup Instructions

cd school-canteen

# Install dependencies
npm install


# Start JSON Server (Mock Backend)
npx json-server --watch db.json --port 5000

👉 Runs API at:

http://localhost:5000

https://aniledzyai.onrender.com


# Start Frontend
npm run dev

👉 Open:

http://localhost:5173

https://aniledzyai.vercel.app/


🔌 API Endpoints (Mocked)
GET /snacks
GET /students
GET /students/:id
POST /students
POST /orders


💡 Key Features Implemented
✅ State management using React Query
✅ Form handling & validation using React Hook Form
✅ Component reusability (SnackCard, OrderForm, etc.)
✅ API integration with Axios
✅ Modal-based ordering system
✅ Dynamic updates (student spending, snack order count)
✅ LocalStorage for recent orders (bonus)
✅ Loading & error states
✅ Clean and responsive UI
🗄️ Mock Data Approach

Used JSON Server to simulate backend APIs.
Data stored in db.json
REST APIs auto-generated by JSON Server
Allows quick prototyping without real backend
